# Enterprise-User-Onboarding-Lab
## 1. Overview

This lab simulates a real-world Level 1 Service Desk onboarding workflow in a Windows domain environment.

The objective was to take a new employee from HR request to fully provisioned domain user with correct access, endpoint baseline policies applied, and documented ticket closure within SLA.

This project demonstrates practical capability in:

Active Directory user and group management

DHCP and DNS configuration

File share permissions (NTFS + Share-level)

Group Policy configuration and deployment

Domain-joined endpoint configuration

ITSM-style ticket handling and documentation
